Memory Walk · York Through Time

Deploy
1) Upload these files to a new GitHub repo, keeping the folder structure.
2) Enable GitHub Pages (Settings → Pages) from main branch / root.
3) Update progress by editing assets/progress.json → change "totalKm".

Notes
- 1 km = 1 year. 0 km = 70 AD.
- Milestones are defined in sketch.js (MILESTONES array).
- Icons are inline SVGs; add more symbols in index.html <defs> if desired.
